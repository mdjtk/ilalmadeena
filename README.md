## 📌 About This Project

This repository contains the frontend code for a business website built with vanilla HTML, CSS, and JavaScript.

### Why Open Source?

This project is open-sourced to:
1. **Showcase Skills** — Demonstrate web development capabilities
2. **Learning Resource** — Help developers understand web development basics
3. **Portfolio Purpose** — A real-world project for developer portfolios

---

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom styling
- **JavaScript** — Interactive elements
- **Deployment** — GitHub Pages / Netlify
